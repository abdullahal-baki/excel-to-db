# Excel to DB Data Import Tool 

### বৈশিষ্ট্য সমূহঃ
১। খুব সহজে CSV এবং XLSX ফাইলের ডাটা ডাটাবেজে ইন্সার্ট করতে পারবেন। 
২। MySQL এবং PostgreSQL এর জন্য টুলটি কার্যকরী। 
৩। Local এবং Remote দুই ধরনের ডাটাবেজের সাথে এইটি কাজ করতে পারে। 
৪। একবার ডাটাবেজ ডিটেইলস দিলে এটি পরবর্তী বার ব্যবহারের জন্য স্বয়ংক্রিয় ভাবে মনে রাখে। 
৫। ইচ্ছা মত টেবিলের নাম দিতে পারবেন। 
৬। স্বয়ংক্রিয় ভাবে ডাটা টাইপ বুঝতে পারে এবং সেই অনুযায়ী Schema সেট করে।.
